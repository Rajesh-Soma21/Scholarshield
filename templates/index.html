System Modules
6-Module Architecture
📥

User Interface

Web form accepting scholarship name, provider, description, URL, email, fee, and amount for analysis.

🗄️

Data Collection

Dataset with 101 labeled records from government portals (NSP, AICTE, UGC) and known fraud patterns.

🧹

Preprocessing

Text cleaning, field combining, TF-IDF vectorization (unigrams + bigrams, 5000 max features).

🔑

Feature Extraction

TF-IDF transforms combined text into numerical feature vectors capturing keyword importance per document.

🧠

ML Models (×4)

Naive Bayes, Logistic Regression, Decision Tree and SVM each independently classify the scholarship.

📊

Prediction & Voting

Majority vote across 4 models produces final verdict (REAL/FAKE/SUSPICIOUS) with confidence score.
